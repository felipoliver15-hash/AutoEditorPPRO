Quando ativo, após o mount o plugin pega o clip da sequência [PRESET]ZOOMIN (ou [PRESET]ZOOMOUT) e copia os atributos (efeitos + keyframes) pras imagens auto-fill — alternando: pares = ZOOMIN, ímpares = ZOOMOUT.

Setup único: crie 2 sequências chamadas [PRESET]ZOOMIN e [PRESET]ZOOMOUT, coloque um clip qualquer em cada uma (Solid Color ou qualquer coisa) e aplique manualmente o preset de zoom desejado (Transform com keyframes de Scale 100→110 etc).

Para cada produto no JSON carregado, gera imagens via Gemini usando png.png como referência visual. Salva como {folder}/gen_1.png, gen_2.png, etc. Usa os textos em image_prompts de cada produto.

Nomeie suas sequências como [TEMPLATE] NomeDoTemplate para que apareçam aqui. Use textos como [[PRODUCT_NAME]] e marcadores como insert:product_image para mapear onde o plugin deve substituir o conteúdo.

Setup rápido do projeto: crie os bins de produto (PROD_1, PROD_2…) com suas mídias e gere as sequências de template vazias já nomeadas. Assim não precisa criar bin nem nomear template na mão.

Cria sequências VAZIAS já nomeadas (pula as que já existem). Você desenha o conteúdo dentro de cada uma depois. Padrão do plugin: [TEMPLATE]PRODUTO, [TEMPLATE]PRECO, [TEMPLATE]LOWERTHIRD, [TEMPLATE]TRANSICAO_1, [TEMPLATE]TRANSICAO_2, [TEMPLATE]LIKE.

Capítulos gerados na montagem (formato do YouTube). Cole na descrição do vídeo. O texto é editável antes de copiar. Marcadores correspondentes são criados na timeline do Premiere.

Relê os marcadores da timeline e atualiza os tempos — útil se você moveu algum marcador de lugar depois da montagem.
